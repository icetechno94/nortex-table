# Nortex-table

Vue component for tables

# Install

    npm i nortex-table
OR

    yarn add nortex-table
    
# Dependents

Component uses:

`vue-i18n` for localization

`vue-multiselect` for filters

`vue-ctk-date-time-picker` for datetime filter

`v-tooltip` for tooltips

`axios` for requests
